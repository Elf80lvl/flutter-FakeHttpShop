# Flutter Fake HTTP Internet Shop

video: https://youtube.com/shorts/4dhpEzIswxY?feature=share

## app features:
* uses bloc 8
* rest api (http, async)
* localization (+ switcher)
* dark and light themes (+ switcher)
* mobile and desktop layouts
* custom icons
* null-safety
